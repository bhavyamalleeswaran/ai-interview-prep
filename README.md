# 🎯 AI-Powered Interview Preparation Assistant

An AI tool that generates interview questions and gives
detailed feedback on your answers — powered by Claude.

## What it does
- Generates interview questions by job role
- Evaluates your answers with a score and feedback
- Tracks your progress over time

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React
- AI: Anthropic Claude API
